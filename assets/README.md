# assets/ (instruções)

Coloque aqui as imagens e ícones usados pelo site.

Estrutura sugerida:
- assets/
  - styles.css
  - script.js
  - images/
    - profile.webp               (sua foto)
    - project-loja.webp
    - project-supermercado.webp
    - project-oficina.webp
    - project-estoque.webp
    - project-erp.webp
  - icons/
    - logo.svg                   (já gerado)
    - apple-touch-icon.png       (gera a partir do logo)
    - favicon-32.png             (gera a partir do logo ou use gerador)

Observações:
- Mantive o tema escuro e a paleta baseada no cartão (verde neon + laranja).
- Se quiser o favicon .ico, use um gerador online (ex.: https://favicon.io/ ou https://realfavicongenerator.net/) e envie os arquivos gerados para assets/icons/.
- Se preferir, me envie a versão em SVG do logo (se tiver) que eu ajusto e gero o conjunto completo de ícones.

Como substituir imagens:
- Substitua os arquivos em assets/images/ mantendo os mesmos nomes, ou atualize os src no index.html.
